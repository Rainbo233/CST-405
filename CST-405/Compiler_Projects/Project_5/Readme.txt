To run the compiler, make sure to input the following commands in the terminal. Ignore the warnings, as long as you execute the following commands, the parser will print the IR code and MIPS code.

$ make clean
$ make parser
$ make parser


The compiler will run the "test.cmm" file, which is the source code used for project 5, and display the compilation time indicating how long it takes to execute the program in the compiler.
